# Bookmarker
Word Bookmarker a simple tool to bookmark specific lines or words of a file and extract it into txt file
# Usage
### Linux/Mac
> pip3 install colorama <br> 
> python3 bookmarker.py

### Windows
> pip install colorama<br>
> python bookmarker.py
